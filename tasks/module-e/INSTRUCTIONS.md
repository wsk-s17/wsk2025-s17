# Module E - Speed Tasks

**Время на выполнение:** 2 часа

Этот модуль проверит вашу способность применить HTML, CSS и знание JavaScript эффективно и творчески. Вам необходимо будет выполнить несколько заданий в течение 2 часов. Будут как простые и рутинные задания, так и более сложные. Обычно на каждую задачу вы можете потратить от 10 до 20 минут.

---

## Описание задач

### **Task 1: HTML/CSS (Easy)**  
Создайте веб-страницу, которая отображает предоставленное изображение (`wheel.png`) и с помощью CSS заставьте его непрерывно вращаться, чтобы казалось, что колесо крутится на странице. Использование JavaScript запрещено — только CSS.

---

### **Task 2: JS (Easy)**  
В файле `script.js` вы найдёте массив с номерами транспортных линий:

```

["ligne n°1", "bus n°24", "tramway n°3", "ligne n°2", "bus n°20", "tramway n°312", "ligne n°130", "bus n°28", "tramway n°20", "ligne n°101"]

```

Необходимо отсортировать линии по возрастанию их номера и вывести результат в консоль браузера.

**Пример результата:**

```

["ligne n°1", "ligne n°2", "tramway n°3", "bus n°20" ...]

```

### **Task 3: HTML/CSS (Medium)**  
Код не предоставлен. Пожалуйста, реализуйте эффект «капель» (looping drip effect), используя только HTML и CSS, опираясь на предоставленное видео. Ознакомьтесь с эталонным видео и создайте такой же эффект.

---

### **Task 4: JS (Medium)**  
Используя JavaScript, создайте интерактивную функциональность для приложения списка задач (to-do list). Пользователи должны иметь возможность добавлять, фильтровать и управлять задачами эффективно, при этом задачи должны автоматически сохраняться для будущих сессий.

**Требования:**
- **Добавление задачи:** возможность добавить задачу с заголовком, описанием и категорией.
- **Фильтрация по категории:** кнопки *(All, Work, Personal, Urgent)*.
- **Отметка выполнения:** переключение состояния задачи с помощью чекбокса.
- **Сохранение задач:** используйте `localStorage`.

HTML и CSS уже предоставлены. Сосредоточьтесь только на JavaScript. Код — в файле `js/todo.js`.

Смотрите примеры и видео `sample.mp4`.

### **Task 8: JS (Hard)**  
Создайте инструмент генерации `box-shadow` с параметрами:

- **x offset:** от `-50px` до `50px`
- **y offset:** от `-50px` до `50px`
- **spread:** от `-50px` до `50px`
- **blur:** от `0px` до `100px`
- **color:** выбор через `color input`

Требования:
- Ползунки (`input type="range"`) для всех параметров, кроме цвета
- Цвет ползунков меняется вместе с выбранным цветом тени
- Кнопка **Copy** копирует значение `box-shadow` в буфер обмена
- Сгенерированный стиль применяется к контейнеру в реальном времени

Ориентируйтесь на видео-пример.

Перетаскивание должно быть плавным, без рывков и мерцаний. Смотрите видео-пример.
